🚀 CONVERGE – Fest Experience Website
An interactive and cinematic tech-fest website built to create an immersive digital experience for visitors.
The website includes dynamic visuals, QR-based entry passes, event schedules, animations, and multimedia sections designed to make a college fest website feel modern and engaging.

🌐 Live Website: https://converge.cicr.in/

✨ Features
🎟 QR Entry Pass Generator
Generates a personalized QR entry pass

Prevents duplicate pass creation using localStorage

Displays a digital ticket with user information

🎬 Cinematic User Interface
Custom animated cursor

Particle and scanline effects

Comic-style UI cards

Smooth hover and animation effects

⏳ Event Countdown
Live countdown timer leading up to the fest.

📅 Dynamic Event Schedule
Editable schedule stored in the browser

Hidden admin editing mode

🎥 Multimedia Experience
Background cinematic video

Intro overlay video with skip option

Interactive visuals and transitions

🤖 Drone Show Section
Animated drone simulation with optional audio controls.

📸 Memories Gallery
Polaroid-style gallery with modal image preview.

🧠 Nexus / Predictions Section
A futuristic themed section showcasing interactive design elements.

📝 Google Form Integration
Registration through a prefilled Google Form redirect.

🛠 Tech Stack
HTML5

CSS3

Tailwind CSS

JavaScript

QR Code JS Library

LocalStorage API

This project is built entirely with frontend technologies.
